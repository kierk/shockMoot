The SwitchBot API doesn't allow direct browser requests due to CORS restrictions. Here are your options:

üöÄ Quick Fix (Recommended)

Install a CORS browser extension:

  1. Install "CORS Unblock" or "CORS Everywhere" from your browser's extension store
  2. Enable the extension for your ShockMoot page
  3. Refresh the page and try again

üîë Alternative: Enable CORS Proxy

If you prefer not to use browser extensions:

  1. Visit: https://cors-anywhere.herokuapp.com/corsdemo
  2. Click "Request temporary access to the demo server"
  3. Return to your ShockMoot page and try again

üíª Local Development with Proxy

For testing without CORS issues using a local proxy server:

  1. Install Node.js from nodejs.org
  2. Open terminal/command prompt in the ShockMoot folder
  3. Run: npm install
  4. Run: npm start
  5. Visit: http://localhost:3000

üêç Simple Local Server

Alternative local testing method:

  1. Open terminal/command prompt in the ShockMoot folder
  2. Run: python -m http.server 8000
  3. Visit: http://localhost:8000
  4. Note: You'll still need a CORS extension for this method
